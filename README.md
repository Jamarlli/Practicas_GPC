# Prácticas GPC

Este repositorio contiene las prácticas de la asignatura Gráficos por Computador (GPC) utilizando Three.js y WebGL.

## Estructura del Proyecto

- `index.html` - Archivo HTML principal
- `js/` - Código JavaScript de las prácticas
- `lib/` - Librerías y dependencias (Three.js, OrbitControls, etc.)
- `models/` - Modelos 3D utilizados en las prácticas

## Tecnologías Utilizadas

- Three.js r140
- WebGL
- OrbitControls
- HTML5/CSS3/JavaScript

## Cómo Ejecutar

1. Clonar o descargar este repositorio
2. Abrir `index.html` en un navegador web moderno
3. También se puede servir desde un servidor web local para evitar problemas de CORS

## Librerías Incluidas

- Three.js (r140)
- OrbitControls
- Cannon.js (física)
- dat.GUI / lil-GUI (interfaz de usuario)
- Tween.js (animaciones)
- Stats.js (estadísticas de rendimiento)
- GLTFLoader (carga de modelos)
- Y otras utilidades de WebGL
